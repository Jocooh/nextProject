## NextStore

###  ✅ Description
맛집, tv에 나왔던 매장들을 위주로 만든 지도 api 프로젝트
---
**🛠️사용 스택**

- react
- next.js
- typescript
- scss
- swr
- react-icons
- firebase(firestore)


---
